### Landing Page Components:
- **HeroSection** — Main banner with CTA button
- **FeaturesSection** — Key capability highlights
- **HowItWorksSection** — 3-step workflow explanation
- **UseCasesSection** — Industry applications (e-commerce, manufacturing, etc.)
- **CtaSection** — Bottom conversion section
- **Footer** — Credits and links
- **Navbar** — Navigation with theme toggle (light/dark mode)

### Demo Page Features:
- **Camera Controls:** Start/stop webcam with device selection, live video preview, calibration
- **Measurement Modes:** 
  1. Auto Detect (Multi-Object)
  2. Line Mode (Point-to-Point)
  3. Polygon Mode (Shoelace Area)
- **Results Panel:** Object list, dimensions (width, height, area, angle, radius), unit conversion, clear/reset
- **Canvas Overlay:** Real-time hover coordinates, distance lines, shapes rendering, centroid markers

### Theme Support:
- **ThemeContext** with localStorage persistence
- Light/Dark mode toggle
